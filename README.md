## ☠️ **Ancaman Siber Paling Umum**

<details>
<summary><b>🎣 PHISHING</b> (Klik untuk detail)</summary>

# Cara Mengenali:
- Email/SMS tidak resmi (contoh: "Klik link verifikasi Bank XYZ")
- Domain website mencurigakan (contoh: "bank-bni.com")
- Meminta data sensitif (password, OTP)

# Contoh Nyata:
💌 Email: "Akun Anda akan dihapus! Segera login via [link palsu]"
📱 SMS: "Anda menang undian! Klaim hadiah di [link scam]"
</details><details> <summary><b>📢 HOAX & DISINFORMASI</b> (Klik untuk detail)</summary>
markdown
# Ciri-Ciri:
- Judul provokatif ("Vaksin bikin mandul!")
- Sumber tidak jelas ("Kata dokter di RS Singapore")
- Dibagikan massal via WA/Telegram

# Tips Verifikasi:
🔎 Cek di situs resmi Kominfo (turnbackhoax.id)
📸 Reverse image search gambar yang dipakai
</details><details> <summary><b>📱 HACKING AKUN</b> (Klik untuk detail)</summary>
markdown
# Modus Operandi:
- Password lemah ("123456", tanggal lahir)
- Social engineering ("Saya dari IT, butuh password Anda")
- Session hijacking (WiFi publik tidak aman)
  
# Proteksi:

🔑 Gunakan 2FA (verifikasi 2 langkah)
🔄 Rutin ganti password (3 bulan sekali)
</details><details> <summary><b>🦠 MALWARE</b> (Klik untuk detail)</summary>

# Jenis-Jenis:

- Ransomware (enkripsi data minta tebusan)
- Spyware (curi data tanpa diketahui)
- Adware (bombardir iklan)

# Pencegahan:

💉 Install antivirus (Windows Defender cukup)
🚫 Jangan klik file .exe tidak dikenal
</details>

🛡️ Tips Proteksi Diri

1. 🔄 Update rutin sistem operasi & aplikasi
2. 🔐 Gunakan password manager (Bitwarden, LastPass)
3. 🌐 VPN untuk akses publik (ProtonVPN gratis)
4. 📱 Backup data 3-2-1 (3 copy, 2 media, 1 offline)
